# summer-school

砧小学校サマースクール向けのプログラミング教材です。

紙の方眼ワークで手作業の大変さを体験してから、同じ作業を命令カード・くり返し・条件に置き換え、プログラミングで楽になることを体験する静的Webアプリです。

## 開発

```sh
npm test
npm run build
```

## Firebase Hosting

Firebase project: `summer-school-kinuta`

```sh
firebase deploy --only hosting --project summer-school-kinuta
```
